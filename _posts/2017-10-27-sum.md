## Description

Given an array of numbers `ns`, return their sum.

```
[1, -1, 2, 7, 9, 124] ‚üπ 142
```


## My solution

```lisp
#lang racket

(define (sum ns)
    (foldr + 0 ns))


(provide sum)
```


## Tests

```lisp
#lang racket

(require rackunit rackunit/text-ui "sum.rkt")
```
